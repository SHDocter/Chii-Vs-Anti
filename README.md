## 使用方法

### 直接使用

下载main.pak并替换掉原文件

### 自定义修改

#### 文件结构

Original -> 官方原文件

Cache -> 正在修改的文件

Modified -> 已修改文件

reanim -> 资源文件和部分配置文件（主要修改的）

particles -> 另一部分资源文件和配置文件

images -> 疑似zombatar资源文件

sounds -> 声音资源

packing&unpaking.pak.with.PVZ.cn.exe -> pak打包解包器 仅支持中文 不保证在其他语种系统中能正常使用

#### 如何修改

替换reanim中的文件，部分需要替换particles

通过打包工具打包成main.pak
